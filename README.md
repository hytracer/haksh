## Installation

There is no public release yet, build from source.
    
## Build from source

Clone the project

```bash
  git clone https://github.com/hytracer/haksh
```

Go to the project directory

```bash
  cd haksh
```

Install dependencies

```bash
  npm install
```

Build haksh

```bash
  npm run build
```
